Dashboard Supermarket

Deskripsi Proyek

Dashboard analitik untuk supermarket yang menampilkan data penjualan, profit, dan inventaris secara real-time. Dibangun dengan React.js untuk frontend dan Node.js untuk backend.

Fitur Utama

  • Visualisasi data penjualan harian, mingguan, dan bulanan
  • Analisis profit margin per kategori produk
  • Prediksi stok barang yang hampir habis
  • Ekspor laporan dalam format PDF dan Excel

Teknologi Digunakan

React.js Node.js MongoDB Chart.js
